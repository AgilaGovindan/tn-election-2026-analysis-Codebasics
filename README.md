# Tamil Nadu Election 2026 — Data Analysis
## Codebasics Resume Project Challenge

### Project Overview
Analysis of 2026 Tamil Nadu Assembly Election results for AtliQ Media.
Story: "TVK Rewrote Tamil Nadu's Political Map in One Election"

### Three Stories
- Q2: The Flip Story — 163 of 234 seats changed hands
- Q3: Vote Share Story — TVK captured 34.92% in debut election
- Q6: Margin Story — 64 seats won with less than 35% vote share

### Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- PowerPoint

### Data Source
Election Commission of India — eci.gov.in
2021 and 2026 Tamil Nadu Assembly Election Results

### Key Findings
- 163 of 234 seats flipped (69.7%)
- TVK won 108 seats in first election
- DMK dropped from 133 to 59 seats
- Chennai Metro: DMK 29 seats → TVK 29 seats (complete handover)
- Average winner vote share: 38.83%
- Only 13 winners crossed 50% vote share
